## 🚀 프로젝트 개요
클라우드 서비스의 비용 부담을 해결하고 실제 서버 운영 및 관리 경험을 얻기 위해 개인 서버를 구축하여 웹 애플리케이션을 배포하는 프로젝트입니다. 이 프로젝트는 서버 성능 최적화 및 보안 강화에 중점을 두어 개발되었습니다.
